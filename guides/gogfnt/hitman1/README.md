### Improved Controls
Download [this .cfg file](./ctrls/Hitman.cfg)

Controls:
	
	Keyboard:
		F1 - Show Laptop
		F2 - Show Status
		2 - Binoculars
		3 - Toggle Mouse Control
		Q - Lean Left
		W - Move Forward
		E - Lean Right
		R - Reload Weapon
		P - Drop Item In Hand
		A - Move Left
		S - Move Backward
		D - Move Right
		F - Empty Hands
		M - Show Map
		LShift - Run
		Z - Turn Left
		C - Turn Right
		LControl - Sneak
		
	Mouse:
		Mouse Wheel Button - Interact
